New Minetest Modpack Project!

We have re-released mishs old MCL2 Minetest Modpack, thus making it a Qyint Project.
We have also created an entirely new Mod.

What do these Mods do?

The Modpack will add quite a lot of Features, Blocks, Enemies and everything else you can think of.
But for now, we only have 3 released Mods. The others are still W.I.P. or dont even exist yet.
These Mods are:
DragonEggLock
This Mod lets you lock a Dragon Egg by placing it in the Crafting Grid.
NonUseless
This Mod adds some Crafting Recipes to make useless Items more useful.
UndoMistakes (NEW)
You can right-click on a Path Block with a Shovel to turn it into Grass again.
Note: Some of these Mods arent even feature-complete, UndoMistakes cant un-strip Wood yet, for example!
You can downlaod the released Mods from ContentDB (ingame) or the experimental Versions from the Github Repository.
