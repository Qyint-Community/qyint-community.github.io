Activision/Blizzard is not quite the best Company when it comes to Linux and FOSS,
so setting up Warcraft III Reforged - and especially Mods like Quenching - hasnt been easy.
This Guide will show you how to install Warcraft III Reforged, as well as Quenching Mod and W3Champions on Linux.
This Post will soon be deleted, as we are planning to create a Warcraft III Ressources Website with all the Information you will ever need. If you have any Questions or Problems, ask us on Revolt or Discord and we might be able to help!

I. Warcraft III Reforged

1. Get Lutris

You can download Lutris via Flatpak. On the Steam Deck you can use Discover for this.

2. Launcher

Install Battle.net through the Lutris Installer. (this script)
(Do NOT use the Warcraft III Reforged Installer! Do NOT log into Battle.net yet, instead, close the Window once youre asked to log in!)



3. Wine Configuration

Before doing anything else, open the Wine Configuration and change the Windows Version to Windows 7



4. Warcraft

Now youre ready to launch Battle.net through Lutris!
(You could also add a Steam Shortcut, which is especially useful for Steam Deck Users. Use SteamGridDB to customize it.)
Download Warcraft III Reforged through Battle.net!
The Game should work as expected!

(...or if you already have the Game installed somewhere on your Computer, copy
C:\Program Files (x86)\Warcraft III
and
C:\Users\[USER]\Warcraft III
to the respective Folders in your new Battle.net Installation Directory. By Default this is $HOME/Games/battlenet/drive_c/...,
then open the Settings and Scan for Games)





II. Quenching Mod

1. Downloads

Visit the Download Page and get both the Client and the Full Version.

Copy both Files to your Warcraft Installation Folder.
(Again, by Default this is $HOME/Games/battlenet/drive_c/Program Files (x86)/Warcraft III/)

2. DLLs (!)

On Linux, if you try to run the Mod Client now, it will just throw a long Error Message at you.
This is because, for some Reason, there are two very important Files missing.
If you copied your Game Files from a Windows/Mac PC, its possible they are there and you can skip this Step.
Dont tell anyone I gave you these Files.
Copy them to the same Directory.

3. Installation

In Lutris, open the Configuration Menu.
(This Menu is very important, so remember how to access it.)

Change the Executable to the QMF2.2.exe you just downloaded.

Now hit "Play".

Two Windows will open, one with the Background Video and one with the actual UI. You can minimize the Background Video one.
(If the Background Video Window appears on top of the UI and you cant move it, try disabling Settings like Gamescope or the Virtual Desktop in the Configuration Window from earlier.)

You can now set your Language to English.

Click Install Mod, then Manual Setup.

Open "My Computer", then C:\Program Files (x86)\Warcraft III\QMF2.2.zip
(The "Client" (the exe File) is the UI you are looking at right now, the "Full Version" is the zip File with the actual Mod Data. You need the Client to configure the Mod. Technically you can download the Mod through the Client, but sometimes it just doesnt work, so this is the most reliable Option.)

Once you see a Window pop up telling you that the Installation was successful, you can close this Menu. (Now you can also delete the zip File.)

4. Configuration

Click Settings, then choose which GPU youre using.


Click Change Theme to set your in-game Background Image. Although I think this is broken on Linux (!)

General


MAKE SURE TO DISABLE THE TERRAIN OPTION ON LINUX!

(I enabled everything else, but do whatever you want.)

Main



Quenchings Trees are pretty high and you might think thats annoying. Here you can reduce the Tree Height.

You can also enable Retro Mode, which are alternative Graphics that you might prefer.

Skin



Here you can select custom Skins for Units and Heroes

Once youre done configuring Quenching Mod, you can close everything, set the Executable back to Warcraft III Launcher.exe in Lutris (see Step 3) and start playing Warcraft!

4.1 Classic Graphics

If you want to use the Classic Graphics, set your Executable back to the Quenching Client. (see Step 3)
Click Settings, then Lock Classic.


Now set the Executable to Warcraft again, start the Game, open the Settings and use the Switch on the bottom-left Side to switch to Classic Graphics.
You will have to do this every Time you switch between Classic and Reforged.

5. More Info (Quenching)

1.

There is a Quenching Mod Guide Book that might help you with some other Issues.

2.

I think the Campaign Menu is useless now that Custom Campaigns have been re-added to Warcraft III Reforged.
(Yes, I say RE-added, the original Warcraft III from 2002 already had Custom Campaigns and Reforged didnt have it until just a few Updates ago...)

3.

You are not logged in to Battle.net if you launch the Game through the Quenching Client.
This is why Im telling you to set the Executable back to the Battle.net Launcher.

III. W3Champions

1. Downloads

Visit the Download Page and download the Windows (Legacy) Version.

Copy the File to your Warcraft Installation Folder. (see II.1)

2. Installation

Set your Executable to the File you just downloaded. (see II.3) And run the Installer. You have to log in to Battle.net! Make sure to re-set your Executable after the Installation - and yes, you can delete the Setup File now!

And thats it! :)
Congratulations, you installed and modded Warcraft III Reforged on Linux!
Have fun!
