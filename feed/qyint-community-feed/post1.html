What is Matrix?

Matrix is an open, federated Protocol that allows anyone on any Server to create an Account and talk to anyone on any other Server.
Our Space is hosted on the "default" Matrix Server, matrix.org. We have created a Matrix Space so more People will switch to Matrix over Discord.

What is Bridging?

A Bot is running on both Matrix and Discord. It will read the Messages sent on either Community and will send them to the other.
This allows Matrix Users to communicate with Discord Users and our Discord Bot, so there are no Downsides to switching to Matrix.
