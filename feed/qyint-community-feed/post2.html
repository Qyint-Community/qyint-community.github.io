Website Update: All Configs Removed

We have decided to remove all of our Config Options.
This includes the German Translations and Dark Mode.
It is simply easier to maintain and edit, the Dark Mode is possible with Browser Extensions and Translations likely wont be used often.

Keys Repo

We have created a new Repository for all of our PGP Public Keys.
(Click here to access the Repo)
We have removed the Keys from mi.sh and the Website, except for a Link.
This way, it is easier to maintain the Keys and to publish your own Keys.
If you want to learn more about PGP Keys, why we use them and how you can upload your own key, read the Repos README.md.
