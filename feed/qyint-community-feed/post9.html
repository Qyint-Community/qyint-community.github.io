Just a couple of Days ago, we have decided to Federate our Revolt Servers.
This means we own more than one Server and People can join whichever one they want.
Everyone will have the same Roles across the different Servers and we all participate in the same General Assemblies.
We have created:
a private Server for General Assemblies
a Chess Server
and a Warcraft Server
along with the original Server.
This wont change a Thing about the original Server,
but People can now join the Servers they want
and we can more easily cater to the Interests of specific Users.
We will also submit every Server (except for the original or private one) to Server Discovery.

You can find all our Server Links in #Info of the Qyint Server.
Please check back later, as we will probably create more Servers in the Future.
